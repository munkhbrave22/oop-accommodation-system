# Accommodation Management System (OOP)

## Overview

This project was developed in Python using Google Colab as part of my Object-Oriented Programming coursework. It models different types of accommodations using object-oriented principles.

## Features

- Base `Accommodation` class
- `Hotel` and `Apartment` subclasses
- Inheritance
- Encapsulation using private attributes
- Getters and setters
- Class variables
- Discount calculation
- Object creation and method interaction

## Technologies

- Python
- Google Colab

## Learning Outcomes

This project strengthened my understanding of core Object-Oriented Programming concepts, including inheritance, encapsulation, constructors, class variables, and writing reusable, modular code.
